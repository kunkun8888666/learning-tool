# 学习辅助工具 · 发布视频脚本（可填模板）

> 用法：把下面所有【】占位符替换成你的真实内容即可。必填项见文末"填空清单"。
> 总时长建议：30–60 秒（短视频）/ 90–120 秒（B站/YouTube 详解版）。

---

## ▶ 视频元信息

- 软件名（占位）：【APP名】
- 主口号（占位）：【口号，例：你的学习数据，只属于你】
- 下载/官网（占位）：【链接】
- 风格：快节奏 + 屏幕录屏 + 关键功能高亮圈选
- BGM：轻快、无歌词；钩子处可停顿 0.5s 加重音

## ▶ 填空清单（开拍前必须替换）

1. 【APP名】——软件正式名（候选：成绩本 / 学迹 / 学盒 / 离线学堂 / 听写机，任选或自定）
2. 【口号】——一句主 Slogan（候选：你的学习数据，只属于你 / 离线也聪明，本地更安心）
3. 【链接】——下载页或官网地址
4. 旁白音色——建议男声沉稳 or 女声轻快，保持一致
5. 录屏分辨率——1080p，系统缩放到 100% 避免模糊

---

## ▶ 分平台微调

- **抖音/小红书（15–30s）**：只保留 #1 #2 #3 #5 #11 #12，节奏更快，字幕加大。
- **B站/YouTube（90–120s）**：把 #3–#10 每屏拉长到 8–10s，加"为什么本地优先"一段口播。
- **应用商店预览视频（20–30s）**：纯功能轮播 #3→#10，不加痛点钩子。
