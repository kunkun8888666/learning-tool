# 学习小工具

> learning-tool 学习辅助工具

## 一、软件特性

成绩追踪：大考小考分开记录，支持 14 种题型小题分值统计，班级、年级排名直观展示；

成绩可视化：本地渲染成绩趋势图表，清晰查看成绩进退步情况，所有数据不上传云端；

AI 智能分析：兼容 OpenAI、Gemini 接口，可自动识别成绩数据、图表、试卷图片并生成学习建议；

文字转语音（TTS）：内置 20 余种中文神经语音，支持自由切换男女声、自定义朗读语速；

单词本：全局快捷键快速取词，统一窗口展示单词释义、音标、配套例句；

听写练习：上传试卷文件→EasyOCR 提取单词文本→AI 语音朗读→支持跟读，完整听写学习闭环；

内置课程平台：一键自动安装依赖，本地部署运行 OpenMAIC 学习资源；

隐私优先：成绩记录、软件配置、试卷图片全部本地存储，无云端同步、无数据上传行为。
